# Casa Alejandro · Reus

Six-language website design for Casa Alejandro, a Catalan Mediterranean restaurant in Reus. Catalan is the default language, with Spanish, English, German, French and Russian localizations.

## Run locally

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## Design

The design direction and source-image status are documented in [docs/design-direction.md](docs/design-direction.md). Full concept screenshots are stored in `design/concepts/`.

The hero, classic-dish, family, and hockey-memory images remain temporary concept artwork; replace those with licensed restaurant photography and original archive material before final launch. The permanent professional-menu destination is `/carta/`; its temporary holding page can be replaced by or redirected to the final PDF when supplied.
